# 4. Enum
## В Python enum (перечисление) — это специальный класс, который используется для создания групп именованных констант. Это полезно, когда нужно представить набор связанных значений, например, дни недели, цвета, состояния и т.д.

## Чтобы создать перечисление, нужно импортировать модуль enum и создать класс, наследующийся от Enum. Вот простой пример:
    from enum import Enum

    class Color(Enum):
        RED = 1
        GREEN = 2
        BLUE = 3

    # Использование перечисления
    print(Color.RED)          # Вывод: Color.RED
    print(Color.RED.name)     # Вывод: RED
    print(Color.RED.value)    # Вывод: 1

    # Перебор элементов перечисления
    for color in Color:
        print(color)

## Основные особенности:Именованные значения: Каждое значение в перечислении должно иметь уникальное имя.Уникальные значения: Значения перечислений могут иметь уникальные значения, хотя это не обязательно.Итерация: Можно перебирать элементы перечисления в цикле.Сравнение: Перечисления поддерживают сравнение как по именам, так и по значениям.Дополнительные возможности:Можно назначать одно и то же значение нескольким именам. В этом случае одно из имен считается "основным", а другие "дубликатами".Перечисления можно использовать как словари, где ключами являются имена, а значениями — соответствующие объекты.
# Дублирование значений
    class Status(Enum):
        PENDING = 1
        IN_PROGRESS = 1
        COMPLETED = 2

    print(Status.PENDING == Status.IN_PROGRESS)  # Вывод: True

## Модуль enum также предоставляет классы IntEnum и Flag, которые обладают дополнительными возможностями, например, можно использовать операции побитового сравнения для элементов перечисления Flag.

# 5. Enumerate
## В Python функция enumerate() используется для одновременной итерации по элементам списка (или другого итерируемого объекта) и для получения индекса каждого элемента. Это очень полезно, когда вам нужно не только перебрать элементы, но и знать их позиции.
    fruits = ['apple', 'banana', 'cherry']

    for index, fruit in enumerate(fruits):
        print(index, fruit)

    Этот код выведет:


    0 apple
    1 banana
    2 cherry

## Как это работает:enumerate() возвращает объект-итератор, который в каждой итерации возвращает кортеж (index, item), где index — это текущий индекс, а item — соответствующий элемент итерируемого объекта. По умолчанию индексация начинается с 0, но вы можете задать другой стартовый индекс, передав его в качестве второго аргумента.

    Пример с начальным индексом 1:

    python
    Копировать код
    for index, fruit in enumerate(fruits, start=1):
        print(index, fruit)
    Этот код выведет:

    Копировать код
    1 apple
    2 banana
    3 cherry
## Применение:Функция enumerate() часто используется, когда вам нужно одновременно итерировать по списку и иметь доступ к его индексам. Это может быть полезно для преобразования элементов на месте, создания нового списка на основе существующего с использованием индексов и в других ситуациях, когда важен порядок элементов.


# 6. Python 
## List - это упорядоченная изменяемая коллекция объектов в Python. Списки могут содержать элементы любых типов данных (например, числа, строки, другие списки) и поддерживают индексацию и срезы.
    # Создание списка
    my_list = [1, 2, 3, 4, 5]

    # Добавление элемента в список
    my_list.append(6)

    # Доступ к элементу по индексу
    print(my_list[2])  # Вывод: 3

    # Изменение значения элемента по индексу
    my_list[1] = 10

    # Удаление элемента
    my_list.remove(10)

    print(my_list)  # Вывод: [1, 3, 4, 5, 6]

## Tuple - это упорядоченная неизменяемая коллекция объектов. Как и списки, кортежи могут содержать элементы любых типов данных, но в отличие от списков, кортежи неизменяемы (их нельзя модифицировать после создания).
    # Создание кортежа
    my_tuple = (1, 2, 3, 4, 5)

    # Доступ к элементу по индексу
    print(my_tuple[1])  # Вывод: 2

    # Попытка изменить элемент (вызовет ошибку)
    # my_tuple[1] = 10  # Ошибка: 'tuple' object does not support item assignment

    # Использование кортежа для возврата нескольких значений из функции
    def get_coordinates():
        return (10.0, 20.0)

    x, y = get_coordinates()
    print(x, y)  # Вывод: 10.0 20.0

## Dictionary - это неупорядоченная коллекция пар "ключ-значение". Ключи в словаре уникальны и могут быть объектами неизменяемых типов данных (например, строки, числа), а значения могут быть любыми объектами.
    # Создание словаря
    my_dict = {'name': 'Alice', 'age': 25, 'city': 'New York'}

    # Доступ к значению по ключу
    print(my_dict['name'])  # Вывод: Alice

    # Добавление новой пары ключ-значение
    my_dict['email'] = 'alice@example.com'

    # Изменение значения по ключу
    my_dict['age'] = 26

    # Удаление пары ключ-значение
    del my_dict['city']

    print(my_dict)  # Вывод: {'name': 'Alice', 'age': 26, 'email': 'alice@example.com'}

    # Перебор ключей и значений
    for key, value in my_dict.items():
        print(f'{key}: {value}')
